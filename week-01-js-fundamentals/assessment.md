## Assessment 2 — Concepts in my own words

**Variable:** A variable is a named container that stores a value. const is for values that don't change, let for values that do. var is the old way and is rarely used today.
**Function:** A function is spcae where we do some logic which we can reuse many times
**Array:** An array is an ordered list of values. Each value has an index starting from 0, so I can access them by position: arr[0], arr[1]."
**Object:** An object is a collection of key-value pairs, where each value has a name (key) and you access it by name: user.name.

## Assessment 3 — HTTP, server, API

**Server:** A server is a computer that runs an app, stores data, and responds to requests from clients (browsers, mobile apps) over the internet.
**API:** An API is a set of endpoints (URLs) and rules that a server exposes, so clients know exactly what they can ask for and how to ask for it."
**HTTP:** HTTP stands for HyperText Transfer Protocol. It's the language clients and servers use to talk to each other on the web."
**HTTP methods:**

- GET — used to get information from the server.
- POST — used to create something
- PUT/PATCH — used to update some information or all object
- DELETE — used to delete
